# Dify app
## type: completion
- 只进行一次问答：没有system prompt
- 支持多模型并发，适合用于模型对比评估



# Knowledge
Data source 只支持
- import from file
- Sync from Notion
- Sync from website (选择爬虫provider)
  - Jina Reader
  - Firecrawl

# [Dify cloud](https://cloud.dify.ai/)

Professional

- up to 50 apps
- Unlimited API rate
